You can use Index through a no-code UI, REST API / clients, and seamless integration with our popular python & typescript framework.

[No-code UI Guide ](../ui)Create, manage, and test indexes using the Index UI. Configure data sources, sinks, and test retrieval without writing code.

[API & Clients Guide ](../api_sdk)Use the Index API and SDK clients for Python and TypeScript. Programmatically manage indexes, upload files, and configure data pipelines.

[Framework Integration ](../framework_integration)Integrate Index with LlamaIndex Python framework. Use LlamaCloudIndex as a drop-in replacement for VectorStoreIndex.

[Index Status Monitoring ](../index_status_monitoring)Best practices for monitoring index health and determining when your index is ready to query using file count endpoints.
